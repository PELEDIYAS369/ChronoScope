# ChronoScope AI

Mission replay, audit, and anomaly detection platform for space operations.

## What It Does

- Deterministic mission replay from real telemetry data
- Tamper-evident audit logging of all mission events  
- Bounded AI anomaly detection with explainable output
- Native CCSDS telemetry format support
- Integrates with COSMOS, OpenMCT, and YAMCS
