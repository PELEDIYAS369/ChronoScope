# Contributing to ChronoScope

Thank you for your interest in ChronoScope AI!

## Important Note

**ChronoScope is a proprietary, closed-source project.** We are not accepting public pull requests or community contributions at this time.

This repository is available for:
- ✅ **Evaluation** — Assess the technology
- ✅ **Demonstration** — Share with stakeholders and investors
- ✅ **Learning** — Understand the architecture and implementation
- ✅ **Feedback** — Report issues and suggest improvements

## How You Can Help

### Report Issues

If you find a bug or have feature suggestions:

1. Check if the issue already exists
2. [Create a new issue](https://github.com/PELEDIYAS369/ChronoScope/issues)
3. Provide:
   - Clear title and description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Your environment (OS, Python version, etc.)

### Share Feedback

For general feedback about ChronoScope:
- **Evaluation feedback:** feedback@chronoscope.ai
- **Feature requests:** features@chronoscope.ai
- **Demo feedback:** demo@chronoscope.ai

### Business Inquiries

Interested in partnership, licensing, or integration?
